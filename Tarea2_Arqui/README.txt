Se desarrollan las tablas de verdad correspondiente al problema, considerando las entradas y salidas.
Si se encuentra un led encendido, se marca con 1, de lo contrario con 0.
Luego se realizan los mapas de Karnaugh para cada salida, agrupando la mayor cantidad de unos posibles,
siempre en potencias de 2.

Para abrir el mapa se necesita de la aplicación Logisim para poder visualizar el archivo de manera esperada.
